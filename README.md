# Personal Research Assistant

An AI-powered scientific literature research assistant that searches academic papers from arXiv, generates structured summaries using Gemini AI, enables PDF-based question answering through RAG (Retrieval-Augmented Generation), and provides an interactive dashboard for researchers and students.

## Project Modules

### Module 4 - PDF Downloader & Parser ✅

Features:
- Download PDF from URL
- Store PDF locally
- Extract text from PDF using PyMuPDF
- Save extracted text into text files
- Generate PDF statistics
- Error handling and testing

### Module 5: Vector Database & Semantic Search ✅

Implemented a vector-based retrieval system using FAISS for efficient similarity search. Text extracted from research papers is converted into vector representations and stored in a searchable index.

**Features:**

* Text chunk storage
* Vector embedding generation
* FAISS vector index creation
* Semantic similarity search
* Retrieval of relevant document passages

**Technologies Used:**

* FAISS
* NumPy
* Python
